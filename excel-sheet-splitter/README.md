# Excel Sheet Splitter Skill

This skill provides functionality to split multi-sheet Excel workbooks into separate files, one file per sheet.

## Installation

1. Ensure Python 3.7+ is installed
2. Install required dependency:
   ```bash
   pip install openpyxl
   ```

## Quick Start

```bash
python scripts/split_excel_sheets.py your_file.xlsx
```

## Features

- Split any Excel workbook (.xlsx, .xlsm) into individual files
- Preserve all formatting, formulas, and cell properties
- Maintain column widths, row heights, and merged cells
- Custom output directory and filename prefix support
- Comprehensive error handling and reporting
